# Intelligent Memory Management Visualizer

An interactive **Operating System project** that simulates and visualizes **dynamic memory management techniques** such as **paging**, **virtual memory**, and **page replacement algorithms** including **FIFO**, **LRU**, and **Optimal**.  
The project provides a modern dashboard-based user interface with **real-time memory frame visualization**, **algorithm comparison**, **event logging**, and **detailed scheduling output**.

---

## 🚀 Features

- 📌 **Paging Simulation**
  - Visualizes how pages are loaded into memory frames
  - Tracks page hits and page faults

- 📌 **Page Replacement Algorithms**
  - **FIFO (First In First Out)**
  - **LRU (Least Recently Used)**
  - **Optimal Page Replacement**

- 📌 **Memory Frame Visualization**
  - Step-by-step frame allocation display
  - Real-time memory state tracking

- 📌 **Memory Access Timeline (Gantt Chart)**
  - Shows page reference sequence visually
  - Helps understand access flow

- 📌 **Algorithm Comparison Dashboard**
  - Compare:
    - Page Faults
    - Hits
    - Hit Ratio

- 📌 **Detailed Output Panel**
  - Displays memory states for each step
  - Shows algorithm execution details

- 📌 **Event Log**
  - Logs page hits, faults, and replacements
  - Makes simulation easier to understand

- 📌 **Modern Premium UI**
  - Dark-themed dashboard
  - Scrollable layout
  - Interactive controls

---

## 🧠 Algorithms Implemented

### 1. FIFO (First In First Out)
Replaces the page that entered memory first.

### 2. LRU (Least Recently Used)
Replaces the page that has not been used for the longest time.

### 3. Optimal Page Replacement
Replaces the page that will not be used for the longest time in the future.  
This gives the **minimum possible page faults** and is used as a benchmark.
